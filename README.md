Инструкция по развёртыванию:

 1. git clone ...
 2. git submodule init
 3. git submodule update
 4. vim -> VundleInstall

Ещё можно установить shfmt:

```
$ sudo apt install golang-go
$ go install mvdan.cc/sh/v3/cmd/shfmt@latest
```
